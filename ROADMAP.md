# Roadmap

This document outlines planned features and improvements for the Quran Multi-Language App.

## v1.1 — Content Expansion
- [ ] Word-by-word translation for all 30 juz
- [ ] Quranic grammar/morphology viewer (root words, i'rab)
- [ ] Hadith references linked to verses (Sahih Bukhari/Muslim)
- [ ] Qira'at support (Hafs, Warsh, Qalun variant readings)
- [ ] Interactive revelation timeline/map

## v1.2 — Cross-Platform
- [ ] Electron desktop app (Windows, macOS, Linux)
- [ ] Chrome extension for quick verse lookup
- [ ] iOS/Android home screen widgets (verse of the day)
- [ ] Flutter mobile app (production-ready rewrite)

## v1.3 — Internationalization
- [ ] Full UI localization (Arabic, French, Turkish, Indonesian, Urdu)
- [ ] Crowdin integration for community translations
- [ ] Full RTL layout toggle for Arabic-speaking users
- [ ] German and Malay translation completion

## v1.4 — Community & Engagement
- [ ] WebSocket live sync (bookmarks/reading position across devices)
- [ ] Discussion forum categories (Q&A, Tafsir, Feature Requests)
- [ ] Newsletter/RSS for project updates
- [ ] Contributor recognition with all-contributors bot

## v1.5 — Performance & Quality
- [ ] Lighthouse CI (automated performance/a11y scoring)
- [ ] Storybook component library
- [ ] Bundle analysis with next-bundle-analyzer
- [ ] Comprehensive unit tests (Vitest + React Testing Library)
- [ ] Visual regression testing

## v2.0 — Advanced Features
- [ ] AI-powered semantic search across translations
- [ ] Personalized reading recommendations
- [ ] Spaced repetition for Hifz mode
- [ ] Community annotations and shared notes
- [ ] Multi-device sync with user accounts
