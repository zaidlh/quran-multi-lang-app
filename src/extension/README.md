# Quran Quick Lookup — Chrome Extension

A Chrome extension for quickly looking up Quran verses from any webpage.

## Features

- Look up any verse by reference (e.g., `2:255` for Ayat al-Kursi)
- See Arabic text with English translation
- Clean, minimal popup interface

## Installation (Development)

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select the `src/extension/` directory

## Usage

1. Click the extension icon in the Chrome toolbar
2. Enter a verse reference in `surah:ayah` format (e.g., `2:255`)
3. Click "Go" or press Enter
